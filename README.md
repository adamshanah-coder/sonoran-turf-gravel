# Sonoran Stone & Turf — Premium Homepage

Next.js homepage rebuild for Sonoran Stone & Turf.

## Run locally

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.

## Before production launch

- Connect the consultation form to a production form endpoint or CRM.
- Add verified contractor license details.
- Replace concept portfolio photography with real project photography as it becomes available.
- Add real same-property before-and-after photography rather than using unrelated stock images.
- Add Privacy Policy and Terms pages after legal copy is approved.
- Compress the large project images during the deployment build or through a CDN.
